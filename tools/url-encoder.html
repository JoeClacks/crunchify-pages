How It Works

URL encoding (percent-encoding) converts characters that are not allowed in URLs into a %XX format, where XX is the hexadecimal value of the character's UTF-8 byte. For example, a space becomes %20 and = becomes %3D. This tool uses the browser's built-in encodeURIComponent / decodeURIComponent APIs and fully supports Unicode text.
